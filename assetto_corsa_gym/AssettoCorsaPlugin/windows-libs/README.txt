- Vjoy.ini -> Vjoy configuration file for Assetto Corsa. Should be copied to C:\Users\%user%\Documents\Assetto Corsa\cfg\controllers\savedsetups
- WASD.ini -> File to control the car using WASD. Should be copied in C:\Users\%user%\Documents\Assetto Corsa\cfg\controllers\savedsetups
- Dlls and Lib folders contain a Python version of socket library, used by the plugin. They should be copied in:
    C:\Program Files (x86)\Steam\steamapps\common\assettocorsa\system\x64
